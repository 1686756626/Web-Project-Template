# 通用经验知识库 — AI Agent 交接手册

> 本仓库不是项目，是所有项目的经验存放地。
> 智能体做新项目或遇到问题时，先来这里查。

## 使用方式

1. 按 `经验/` 目录下的分类查找对应经验
2. 每篇经验格式：**场景 → 规则 → 具体做法**
3. 照着做就行，不用自己试错

## 经验分类

| 目录 | 内容 |
|------|------|
| `前端/` | SPA路由、Tailwind深色模式、打印样式、移动端适配 |
| `部署/` | Nginx静态站点、GitHub API反代、缓存破坏 |
| `GitHub/` | 中文文件推送变空（红线）、中文路径raw返回HTML |
| `架构/` | SPA标准结构、localStorage存储、聊天面板组件 |
| `教学/` | 先练后讲、时间管理、出题流程、课后同步 |
| `数据/` | students.json课表格式 |

## 贡献新经验

遇到新的通用经验时：
1. 确认经验不属于单一项目，而是跨项目通用的
2. 在对应子目录下新建 md，文件名用中文描述问题
3. 格式：**来源项目 → 规则 → 具体做法 → 注意事项**
4. commit 时注明来自哪个项目

## 和各项目的关系

| 项目 | 仓库 | 经验来源 |
|------|------|----------|
| Teaching-Dashboard | 教学系统前端 | SPA路由、GitHub API、打印、深色模式、缓存 |
| Finance-Manager | 个人财务系统 | localStorage、登录验证、聊天面板 |
| Andy-Shihundan | 搞笑网站 | 轻量SPA、JSON数据源 |
| CXL | 高考冲刺教学 | 多科目教学、cron定时任务、时间管理、出题 |
