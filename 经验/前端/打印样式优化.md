# 打印样式优化

> 来源：Teaching-Dashboard 项目

## 问题

浏览器打印时会把导航栏、按钮、弹窗装饰全部打出来，字号不合适，内容跨页断裂。

## 解决方案

```css
@media print {
    /* 隐藏不需要打印的元素 */
    body { background: white !important; font-size: 11pt; }
    nav, #toast-container, .no-print, button { display: none !important; }
    #global-search { display: none !important; }

    /* 弹窗打印模式 */
    #preview-modal { position: static !important; background: white !important; }
    #preview-modal > div:first-child { display: none !important; }  /* 遮罩层 */
    #preview-modal > div:last-child { position: static !important; border-radius: 0 !important; }

    /* Markdown 内容优化 */
    .md-preview { font-size: 11pt; line-height: 1.8; }
    .md-preview h1, .md-preview h2, .md-preview h3 { page-break-after: avoid; }
    .md-preview tr, .md-preview li { page-break-inside: avoid; }
    .md-preview blockquote { background: #f9fafb !important; }

    /* 页边距 */
    @page { margin: 1.5cm; }
}
```

## 注意

- `-webkit-print-color-adjust: exact` 让背景色在打印时保留
- 给不需要打印的按钮加 `no-print` class
- `page-break-inside: avoid` 防止表格行或列表项跨页断裂
