# 打印样式

> 来源：Teaching-Dashboard

## 规则

打印时只出内容，隐藏所有导航和按钮。用 `@media print` 控制。

## 具体做法

在 index.html 的 `<style>` 中加：

```css
@media print {
    body { background: white !important; font-size: 11pt; -webkit-print-color-adjust: exact; }
    nav, #toast-container, .no-print, button { display: none !important; }
    #global-search { display: none !important; }

    /* 弹窗打印模式：去掉遮罩，内容静态展示 */
    #preview-modal { position: static !important; background: white !important; }
    #preview-modal > div:first-child { display: none !important; }
    #preview-modal > div:last-child { position: static !important; border-radius: 0 !important; box-shadow: none !important; }

    /* Markdown 内容 */
    .md-preview { font-size: 11pt; line-height: 1.8; }
    .md-preview h1, .md-preview h2, .md-preview h3 { page-break-after: avoid; }
    .md-preview tr, .md-preview li { page-break-inside: avoid; }
    .md-preview blockquote { background: #f9fafb !important; }

    @page { margin: 1.5cm; }
}
```

## 注意

- 不需要打印的按钮加 `no-print` class
- `page-break-inside: avoid` 防止表格行跨页断裂
- `page-break-after: avoid` 防止标题后紧跟分页
