# 移动端适配

> 来源：Teaching-Dashboard 项目

## 问题

纯前端 SPA 在手机上导航栏拥挤、内容溢出、弹窗不适配。

## 解决方案

### 1. 导航栏图标化

小屏只显示图标，不显示文字：

```html
<a href="#dashboard" class="nav-link">
    <i class="fa-solid fa-chart-pie mr-1.5"></i>
    <span class="hidden sm:inline">仪表盘</span>  <!-- 小屏隐藏文字 -->
</a>
```

Tailwind 的 `hidden sm:inline` 在 `≥640px` 才显示文字。

### 2. 响应式网格

```html
<div class="grid grid-cols-2 lg:grid-cols-4 gap-4">
```

小屏 2 列，大屏 4 列。

### 3. 弹窗全屏

```css
@media (max-width: 640px) {
    #preview-modal > div:last-child { inset: 0 !important; border-radius: 0 !important; }
}
```

### 4. 内容区紧凑

```css
@media (max-width: 640px) {
    #content { padding: 0 8px; }
    .md-preview table { font-size: 12px; }
    .md-preview th, .md-preview td { padding: 4px 6px; }
}
```
