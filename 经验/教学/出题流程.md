# 出题流程

> 来源：Teaching-Dashboard、CXL

## 规则

先出答案再出题。出完自查。不考没教过的。

## 具体做法

### 流程

1. 确定考试范围（查 Teaching-Calendar 或教学日历）
2. 把正确答案先写出来
3. 出题
4. 自查清单：
   - 所有知识点全覆盖了吗？
   - 答案对吗？
   - 题目搭配标准吗？
   - 有没有考没教过的内容？
5. 确认无误再发给学生

### 题目围绕下次考试出

从教学日历确认下次考试范围 → 考什么练什么。不按兴趣出题。

### 简答题分支判断

预判学生的具体错误写法。不只是"答错"两个字，而是：

```
分支1：她写了"态度变了" → 说明停留在表面行为差异，没看到情感内核
分支2：她写了"前后矛盾" → 混淆了行为和态度
分支3：正确答出"态度不变，只是表达方式不同" → 肯定+追问为什么
```

## 注意

- 薄弱点必须出对应的专项练习，不能只在方法总结里提一句
- 学生说"会了"不等于真会了，要多轮交叉测试确认
