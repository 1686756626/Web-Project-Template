# 浏览器缓存破坏

> 来源：Teaching-Dashboard（白屏事故根因之一）

## 规则

每次部署 app.js 后必须递增版本号，否则浏览器用缓存旧版。

## 具体做法

```html
<script src="app.js?v=20260404l"></script>
```

每次部署时递增：

```bash
sed -i 's/app.js?v=[^"]*/app.js?v=20260404m/' /var/www/xxx/index.html
```

## 注意

- 源码目录和部署目录的版本号必须同步
- **`<script>` 标签必须完整闭合**：曾因末尾 `<script src="app.js?v=xxx"` 缺少 `"></script>` 导致 JS 从未加载，白屏
- 部署后刷新浏览器确认版本号生效
